## Things needed to run bot

- Need [**NodeJS**](https://nodejs.org/en/) version **v16.6 or higher**
- Need [**Visual Studio Code**](https://code.visualstudio.com)

- In `./config.json` change prefix and token

- `npm install` to install pack

- `node .` run the bot
